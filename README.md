# gs-poll
一个简单的MEAN投票示例。
<br>参考URL： http://www.ibm.com/developerworks/cn/web/wa-nodejs-polling-app/index.html?ca=drs-&utm_source=tuicool

<h1> 运行准备 <h1>
<p> 安装nodejs</p>
<p> 安装express-generator</p>
<p> 安装MongoDB</p>
<h1> 运行 <h1>
<p> git clone https://github.com/kuangcaibao/gs-poll</p>
<p> cd gs-poll </p>
<p> npm install </p>
<p> node app.js </p>
